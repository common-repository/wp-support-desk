WP Support Desk is a full-featured WordPress plugin to accept, manage and respond to customer support tickets all from a simple and unified dashboard

== Description ==
Most of the WordPress support plugins available online create huge tables on the connected database which can slow down your site. Beware from these resource hungry support plugins. Our WP Support Desk plugin solves this problem. We’ve used efficient algorithm in WP Support Desk so that your site and system will remain speedy. 

WP Support Desk doesn’t make your site slow. You will love this fully Ajax based simple and lightweight plugin. WP Support Desk is very easy and fun to use for both the general users and admins. Install and activate WP Support Desk now to start offering a smoother and better support mechanism to your customers.


= Features =

  * Fully ajax based page load
  * Ticket priority set
  * Knowledge base module with category
  * Multilingual Ready
  * User Login form
  * Unlimited Custom ticket field
  * Departments
  * Ajax ticket search
  * Ticket validation
  * Knowledgebase article voting


= Features (More in the pro version) =
  * Send attachment with reply or creating new ticket (Add-on)

  * Many more feature coming soon

== Installation ==

1. Go to Dashboard > Plugins > Add New, then search WP Support Desk and click Install Now
2. Upload 'wp-support-desk' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Click on the new menu item "WP Support Desk" and Setup your Settings.

== Frequently Asked Questions ==

= Q. Where can I get support? =
A. You can get support by posting on the support section of this plugin on WordPress plugins directory or directly mailing to support@themeum.com

= Q. Can I use my existing WordPress theme? =
A. Sure, you can use your existing WordPress theme with WP Support Desk, we will use just a page for display user panel.

= Q. Where can I report a bug? =
A. Found a bug? Please let us know by posting on the support section of this plugin on WordPress plugins directory or directly mailing to support@themeum.com

= Q. Is WP Support Desk free? =
A. There are two versions of WP Support Desk. One is free and another is paid. The paid version has some more advanced features which are not accessible in the free version.
